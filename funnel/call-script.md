# Cold Call Script — SMB Workflow Triage

## Opener (15 seconds)
"Hey [Name], quick one — we help small businesses save 5–10 admin hours a week by fixing one repetitive workflow around tools they already use (Square/Stripe/Clover, CRM, spreadsheets)."

## Permission
"Bad time, or got 30 seconds?"

## Qualifier questions (pick 2–3)
- "Any daily copy/paste between systems right now?"
- "Who owns end-of-day reporting?"
- "If that person is out, does work pile up?"

## Value bridge
"Got it. That’s exactly what we fix in a 7-day sprint — no platform rip-and-replace."

## CTA
"Want to do a 15-minute Workflow Triage this week? We’ll map one bottleneck and give you a clear fix plan."

## Objection handling
- "We already have software." → "Perfect — we keep your stack and automate around it."
- "No budget." → "Understood. We start with one workflow so ROI is visible fast."
- "Send info." → "Happy to. What’s the best email, and should we hold a 15-min slot tentatively?"
